# Todolist Simple Web

todolist web app is simple web app application todolist

## Installation

Use the package manager [git](https://github.com/dyaksa/simple-todolist.git) to install todolist.

```bash
git clone https://github.com/dyaksa/simple-todolist.git
```

## Usage

```python
node app.js
```
